Hello! You are reading the brief documentation for the second stage of the streaming platform development, specifically the creation of a simple client-server application.

To begin, compile the two files located in the stream_client and stream_server folders using the Linux terminal (Bash).

Go to the root folder and enter the following commands:

g++ client.cpp -o client.exe

g++ server.cpp -o server.exe

Further steps regarding the individual startup of the server and client can be found in the README_Client and README_Server files within the stream_client and stream_server folders, respectively.
